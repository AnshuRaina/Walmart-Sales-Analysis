The file contains the following
-------------------------------

store -  Id of store

dept - Id of dept

Train Date - Date of train data taken from Nov, Dec 2011 and Jan-Jul 2012

isHoliday - Flag whether it is holiday or not

Test Data - Date of test data taken from Nov, Dec 2012 and Jan-Jul 2013

Linear - Weekly sales prediction using linear regression algorithm

SMO - Weekly sales prediction using SMOreg algorithm

Linear difference

Smoreg difference


Mean Absolute Error
-------------------

It contains the mean absolute error of Linear regression and SMOreg algorithm
